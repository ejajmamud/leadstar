# Getting Started with LeadStar

## Installation

1. Download from Chrome Web Store or load unpacked
2. Navigate to `chrome://extensions`
3. Enable Developer mode
4. Click "Load unpacked" and select the extension directory

## Initial Setup

### Supabase Configuration
1. Create a Supabase project
2. Set Site URL: `https://apps.ejaj.space`
3. Set Redirect URL: `/leadstar/auth`
4. Copy anon key to extension settings

### First Extraction
1. Open side panel (click extension icon)
2. Navigate to any http/https website
3. Choose extractor tool
4. Configure extraction parameters
5. Run extraction

## License Verification

The extension automatically checks license status:
- Cached for 30 minutes
- 24-hour offline grace period
- Falls back to free after grace
